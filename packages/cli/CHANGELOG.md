# opscale

## 0.1.1

### Patch Changes

- Add local `opscale config init` support so users can save database credentials on their own machine instead of pasting DSNs into AI chat.

## 0.1.0

### Minor Changes

- Initial public release of Opscale as a single AI agent skill and CLI for read-only operations analytics over SQL databases.

  Includes the Opscale CLI, one language-adaptive AI agent skill installable through the generic `skills` installer, Node.js database drivers for PostgreSQL/MySQL/SQLite/SQL Server, SQL guardrails, bilingual documentation, and open-source project metadata.
