# opscale-drivers

Node.js database drivers for Opscale.

Supported execution targets:

| Database | Node package |
| --- | --- |
| PostgreSQL | `pg` |
| MySQL / MariaDB | `mysql2` |
| SQLite | `sql.js` |
| SQL Server | `mssql` |

Use this package through `opscale` unless you are embedding Opscale in
another Node.js tool.
